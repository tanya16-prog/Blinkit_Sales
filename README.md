# Blinkit_Sales

1. Project Title / Headline :-
2. 
       Blinkit Grocery Sales Analysis Dashboard | Power BI Business Intelligence Project

   
2 . Short Description / Purpose :- 

      The Blinkit Grocery Sales Dashboard is an interactive Power BI project designed to analyze sales performance, customer purchasing behavior, and product insights. It provides a comprehensive view of key business metrics, enabling stakeholders to monitor sales trends, outlet performance, item categories, customer ratings, and inventory distribution.

The dashboard supports data-driven decision-making by presenting business KPIs through interactive visualizations and filters

 5.  Data Source :-
 6.  
      The dashboard is built using Blinkit grocery sales transaction data containing information such as:
        Product Details,  Item Type,   Item Fat Content, Outlet Type, Outlet Size, Outlet Location, Outlet Establishment Year,
         Sales Amount, Customer Rating, Item Visibility, Total Items Sold
